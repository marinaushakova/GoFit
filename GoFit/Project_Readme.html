To set up:
1) Clone project from bitbucket
2) Right click on solution in solution explorer and select rebuild solution
3) If there are missing assemblies right click on the GoFit project and select Manage NuGet Packages.
4) Install EntityFramework
5) Install Microsoft ASP.NET MVC
6) Right click on GoFit.Tests project in the solution explorer and select Manage NuGet Packages.
7) Install EntityFramework
8) Install Microsoft ASP.NET MVC
9) Rebuild the project and all assembly warnings should be resolved
To test:
Click run all from the Visual studio test explorer
